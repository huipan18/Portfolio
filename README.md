# 💻 Macintosh-OS

A web-based remake of the classic Macintosh desktop, built as an interactive portfolio using modern tech.

---

## 🚀 Live Demo  
👉 https://portfolio-green-theta-33.vercel.app/

---

## 🛠️ Tech Stack  
- Next.js  
- Tailwind CSS  
- TypeScript  
- Vercel  

---

## ✨ Features  
- Classic Mac-style UI  
- Fully responsive  
- Interactive desktop experience  
- Portfolio integration
